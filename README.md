# DocCare - Doctor Booking App 🏥

![App Preview](home-screen.png)

## 📌 Overview
DocCare is a UI/UX design project for a doctor appointment booking mobile application designed using Figma. The goal is to simplify the process of finding doctors and booking appointments online.

## ❗ Problem
Users often face difficulty in finding the right doctor, checking availability, and booking appointments quickly in traditional healthcare systems.

## 💡 Solution
Designed a user-friendly and clean interface that allows users to easily search doctors, view availability, and book appointments seamlessly.

## 🔄 User Flow
User opens app → searches doctor → views profile → selects time slot → books appointment → receives confirmation

## 🎨 Design Process
- User Research
- Wireframing
- UI Design in Figma
- Prototyping

## ✨ Features
- Doctor search by specialization
- View doctor profiles
- Appointment booking
- Time slot selection
- Booking confirmation

## 🛠 Tools Used
- Figma
